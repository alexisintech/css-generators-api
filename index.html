CSS Generator API

Provides a directory of websites that generate different CSS design elements.

Search for a CSS generator by relevant keywords.

Documentation

Each website has the following information stored as an object.

There are 5 properties/keys that come with each website's object:

PROPERTY DESCRIPTION OF PROPERTY
name[String] Name or title of the website.
image[String] Image of the website.
url[String] Link to website.
url[String] A brief description about what the website can generate.
keywords[Array] Relevant keywords related to the generator.

Endpoints

Each website object is stored in the websites array.

Direct link to the entire websites array:

https://cssgenerator.com/api

*The "/api" endpoint will return every website object in the array.

Link to filtered objects that contain a specified keyword:

https://coding-websites-api.herokuapp.com/api/<keyword>

*The "/api/keyword" endpoint accepts a keyword that will be used to search through the websites array.

Example code-block of how to fetch with JavaScript:

const URL = `https://cssgenerator.com/api/${keyword}`; fetch(URL)    .then(res => res.json())    .then(data => console.log(data))    .catch(err => console.error(err));
